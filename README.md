# Playful Husbandry
## What is this? 
Personally, I think Minecraft's breeding is very drab. Feed two animals, another one pops out. Okay. Sure. Whatever. It's fine. 
But what if it was more?
That's where Playful Husbandry comes in! This mod adds a few new items and a new "Animal Genetics" system that makes you think critically about which animals you breed and when.
